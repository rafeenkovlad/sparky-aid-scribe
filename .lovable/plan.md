## Цель
Заменить модалку `PhotoAnnotator` на режим «чат с фотографией»: при тапе по фото из коллажа чат не открывает оверлей, а переключается в специальный фокус‑режим, где сверху закреплено фото, рядом — быстрые действия (элемент, теги по разделам «Серьёзные/Мелкие»), а нижний композер чата становится «полем заметки к фото». Отправка из композера записывает заметку, а через ИИ может породить новый тег и положить его в правильный раздел (serious / non_serious) с учётом текущего вердикта.

## UX

```text
┌─ Header чата ─────────────────────────────┐
│ ← Назад к разделу «Кузов»     ⋯ Удалить   │
├───────────────────────────────────────────┤
│   [ Большое фото · pinch‑zoom · swipe ]   │
│         •  •  ●  •  •   (2 / 5)           │
├───────────────────────────────────────────┤
│ Элемент: [Бампер пер.] [Капот] [Крыло…]   │
│ Вердикт: ( OK )( Мелкие )( Серьёзные )    │
│                                           │
│ Серьёзные:  ✓ Скол  · Трещина · Вмятина… │
│ Мелкие:     ✓ Царапина · Потёртость …    │
│ ✨ Новые: «Скол краски ×»                  │
│                                           │
│ 💬 Заметка (через композер ниже):         │
│   «Скол на левом углу, до металла»        │
└───────────────────────────────────────────┘
┌─ Композер чата (фиксированный) ───────────┐
│ 📎 Фото · бампер пер.        [Снять режим]│
│ [ ...пишите заметку к фото...        ] 🎤 ↑│
└───────────────────────────────────────────┘
```

- Тап по миниатюре в `InspectionCollage` не открывает модалку — переключает ChatApp в `photoFocusMode = { photoIdx }`.
- Лента сообщений в фокус‑режиме скрыта, вместо неё рендерится `PhotoFocusView`.
- Композер остаётся тот же (привычная клавиатура, голос, отправка), но:
  - над ним появляется чип «📎 Фото · {элемент}» с крестиком «снять режим»;
  - placeholder меняется на «Заметка к фото…»;
  - отправка стрелкой = сохранить заметку (+ опционально распознать новый тег ИИ, см. ниже).
- Свайп влево/вправо листает фото внутри текущего раздела; pinch‑zoom через простой `transform: scale` или нативный `<img>` в `object-contain` + двойной тап на 2×.
- «Назад» = выход из фокус‑режима, возврат к ленте с обычным скроллом к коллажу.

## Быстрые действия (без модалки)

- Элемент: горизонтальный скроллер чипов из `section.elements`. Тап = смена `photo.elementId` и `finding`‑ключа.
- Вердикт: три чипа OK / Мелкие / Серьёзные (та же палитра emerald/amber/rose).
- Теги: два блока с заголовками **Серьёзные** и **Мелкие**, чипы из каталога `loadSectionTags(section)`. Тап ставит/снимает; при выборе тега из «Серьёзные» при `verdict==="ok"` вердикт авто‑поднимается в `serious` (как сейчас в аннотаторе), аналогично для мелких → `minor`. Это и есть «вес» тега в смысле текущей доменной модели (`type: "serious" | "non_serious"`).
- Pending‑теги отображаются отдельной строкой «✨ Новые». Кнопка «＋ Свой тег» — инлайн‑инпут как сейчас.

## Композер как «заметка к фото»

Состояние `photoFocusMode` живёт в `ChatApp`. Пока оно активно:
- `composer` value/onChange разделяется: используется отдельное поле `photoNoteDraft` (инициализируется из `finding.note`), чтобы не путать с обычным сообщением треда.
- Сабмит из композера:
  1. Сохраняет `note = photoNoteDraft.trim()` в `finding` через `applyAnnotator`‑подобный патч (без закрытия экрана).
  2. Если в тексте присутствует ключевое слово, которого нет среди уже выбранных тегов, и пользователь нажал отдельный экшен **«✨ Распознать тег из заметки»** (кнопка‑искра рядом с микрофоном) — вызываем `analyzeInspectionPhoto` с подсказкой `noteHint`, мёрджим только новые `seriousTagIds` / `noSeriousTagIds` / `pendingTags` (note не перезаписываем). Существующий тег попадает в правильный раздел по его `type`; pending‑тег создаётся с `severity`, равной активному вердикту, либо `serious`, если в подсказке встретилось серьёзное слово из `lexicon`.
- Голосовой ввод работает как обычно — попадает в `photoNoteDraft`.
- Кнопка «Снять режим» (или ← в хедере) возвращает обычный композер.
